Summary: Latitude vs. X

The purpose of this project was to Determine the ideal Vacation Spot for a set of criteria that we determined would be our ideal Temperature. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.


After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude.


Factors we looked at included:

  • Temperature
  • Cloudiness
  • Wind Speed
  • Humidity
This site provides

Data and visualizations, that were generated in that process of the analysis, as well as explanations and descriptions or correlations witness in the gathering process.
